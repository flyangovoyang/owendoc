Anaconda

虚拟环境

conda create -n  python=3.5  # 新建虚拟环境
conda activate 
conda deactivate

更换anaconda安装源

下面是通过命令行的方式添加，还可以通过修改配置文件的方式。

conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main
conda config --set show_channel_urls yes

Linux Bash

vim的移除和重新安装

sudo apt-get remove vim-common
sudo apt-get install vim

scp

文件复制

# file
scp local_file remote_username@remote_ip:remote_folder
scp local_file remote_username@remote_ip:remote_file
# directory
scp -r local_folder remote_username@remote_ip:remote_dir

wc

统计文件内容，有三个选项，不指定的话默认全部采用。

wc -c   # 统计字节数
wc -l   # 统计行数
wc -w   # 统计词数

安装flashplayer

下载压缩包，然后解压成/usr/lib/firefox/browser/plugins放在火狐浏览器的插件目录下，该目录为/usr/lib/firefox/browser/plugins

安装jdk

  1. 官网下载jdk包，现在需要翻墙注册账号，越来越恶心了。
  2. 在/usr/local/目录下新建一个文件夹java
  3. 将压缩包复制到java后解压
  4. 然后删除安装包，并配置环境变量 
    sudo vi /etc/profile
# 或者
sudo vi ~/.bashrc
    配置内容为： 
    JAVA_HOME=/usr/local/java/jdk1.8.0_161
CLASSPATH=$JAVA_HOME/lib/
PATH=$PATH:$JAVA_HOME/bin
export PATH JAVA_HOME CLASSPATH
  5. 保存生效 
    source /etc/profile
# 或者
source ~/.bashrc
