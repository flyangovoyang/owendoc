Python Collections

返回首页   create: 2020-08-28 09:59:55

collections模块是对python内置数据类型（如list, tuple, dict, set）的补充，包含了几个比较实用的类。

  • named_tuples
  • deque
  • ChainMap
  • Counter
  • defaultdict
  • OrderedDict
  • UserList
  • UserDict
  • UserString

1. defaultdict

defaultdict是dict的子类，通过传递一个默认工厂函数来创建默认的value

import collections

d = collections.defaultdict(list)
print(d['a'])  # []

2. OrderedDict

OrderedDict也是dict的一个子类，因为一般情况词典的key是无序的，但是OrderedDict的key是有序的，它会记住每个key被加进去的先后次序，从而保持key的顺序。
