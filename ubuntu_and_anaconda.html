Anaconda

虚拟环境

conda create -n  python=3.5  # 新建虚拟环境
conda activate 
conda deactivate

更换anaconda安装源

下面是通过命令行的方式添加，还可以通过修改配置文件的方式。

conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main
conda config --set show_channel_urls yes

Linux Bash

vim 文本编辑

移除和重新安装
sudo apt-get remove vim-common
sudo apt-get install vim
配置

给当前用户配置，在文件~/.vimrc中配置，配置的注释符不是#而是"

set encoding=utf-8
set tabstop=4
set autoindent
set expandtab
set number
"set cursorline
set termencoding=utf-8

netstat 查看端口

首先安装

apt-get install net-tools

查看端口占用情况

netstat -anp

scp

文件复制

# file
scp local_file remote_username@remote_ip:remote_folder
scp local_file remote_username@remote_ip:remote_file
# directory
scp -r local_folder remote_username@remote_ip:remote_dir

wc

统计文件内容，有三个选项，不指定的话默认全部采用。

wc -c   # 统计字节数
wc -l   # 统计行数
wc -w   # 统计词数

更改ubuntu软件镜像

首先查看系统版本和内核情况

uname -a

然后备份镜像源

sudo cp /etc/apt/sources.list /etc/apt/sources.list.backup

对于18.04 bionic，使用如下镜像源

deb http://mirrors.aliyun.com/ubuntu/ bionic main restricted universe multiverse
deb-src http://mirrors.aliyun.com/ubuntu/ bionic main restricted universe multiverse

deb http://mirrors.aliyun.com/ubuntu/ bionic-security main restricted universe multiverse
deb-src http://mirrors.aliyun.com/ubuntu/ bionic-security main restricted universe multiverse

deb http://mirrors.aliyun.com/ubuntu/ bionic-updates main restricted universe multiverse
deb-src http://mirrors.aliyun.com/ubuntu/ bionic-updates main restricted universe multiverse

deb http://mirrors.aliyun.com/ubuntu/ bionic-proposed main restricted universe multiverse
deb-src http://mirrors.aliyun.com/ubuntu/ bionic-proposed main restricted universe multiverse

deb http://mirrors.aliyun.com/ubuntu/ bionic-backports main restricted universe multiverse
deb-src http://mirrors.aliyun.com/ubuntu/ bionic-backports main restricted universe multiverse

随后需要更新

apt-get update
apt-get upgrade

用户管理

修改用户密码

sudo passwd owen

安装flashplayer

下载压缩包，然后解压成/usr/lib/firefox/browser/plugins放在火狐浏览器的插件目录下，该目录为/usr/lib/firefox/browser/plugins

安装jdk

官网下载jdk包，现在需要翻墙注册账号，越来越恶心了。

在/usr/local/目录下新建一个文件夹java

将压缩包复制到java后解压

然后删除安装包，并配置环境变量

sudo vi /etc/profile
# 或者
sudo vi ~/.bashrc

配置内容为：

JAVA_HOME=/usr/local/java/jdk1.8.0_161
CLASSPATH=$JAVA_HOME/lib/
PATH=$PATH:$JAVA_HOME/bin
export PATH JAVA_HOME CLASSPATH

保存生效

source /etc/profile
# 或者
source ~/.bashrc
